App Description

Welcome to the Hellometer data viewing app! This app takes a csv file with columns arrival_time (in seconds), dettection_id, pre_orderboard (in seconds), orderboard (in seconds), pickup, and total_time (in seconds). You can drag your csv file to upload the data and histograms for each order fulfillment step will be shown for the week, and on average for each hour of the day.
